# File untuk Upload ke InfinityFree

Folder ini berisi file-file yang KHUSUS untuk upload ke InfinityFree.
**JANGAN dipakai di local development!**

## File:

### 1. `index.php`
Upload ke `/htdocs/` di InfinityFree (replace index.php yang ada)

### 2. `.htaccess`  
Upload ke `/htdocs/` di InfinityFree

### 3. `.htaccess-private`
Upload ke `/private_html/` di InfinityFree, rename jadi `.htaccess`
